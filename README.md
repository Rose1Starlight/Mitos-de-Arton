# Mitos de Arton v0.7

Um módulo não-oficial para **Foundry VTT v13** contendo conteúdo do suplemento **Mitos de Arton**, adaptado para uso direto nas mesas de **Tormenta20**.

Este projeto tem como objetivo facilitar a vida de mestres e jogadores, trazendo o material do suplemento para dentro do Foundry em formato de Compendio pronto para arrastar e usar.

##  Aviso importante

Este é um projeto **fanmade e não-oficial**.
O suplemento original pertence ao autor do homebrew e o sistema Tormenta20 pertence à Jambô Editora.

Este módulo **não substitui o material original** — você ainda precisa do PDF para entender as regras completas e o contexto do conteúdo.

Este Modulo ainda não possui as Automoções, apenas as partes escritas dos poderes e habilidades

## O que é este módulo?

Este módulo adiciona ao Foundry:

* Compendio organizado
* Conteúdo pronto para arrastar para a ficha
* Integração com o sistema Tormenta20
* Padronização para uso em mesa online

## Requisitos

* Foundry VTT **v13**
* Sistema **Tormenta20** instalado

## Instalação

1. Baixe o módulo colando o link no foundry
```
https://raw.githubusercontent.com/Rose1Starlight/Mitos-de-Arton/main/module.json
```
1. Ou baixe os arquivos e cole na pasta

```
FoundryVTT/Data/modules/
```

2. Ative o módulo no seu mundo
3. Os compendiums aparecerão automaticamente 🎉

## Objetivo do projeto

Este módulo existe para:

* Facilitar o uso do homebrew em mesas online
* Incentivar a comunidade Tormenta20
* Tornar suplementos feitos por fãs mais acessíveis
* Aprender e praticar criação de módulos Foundry
